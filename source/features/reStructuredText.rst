- Simple text markup
- Stylistic tags
   - **bold**
   - *italics*
   - lists
   - multiple heading levels
   - tables
- Semantic tags
   - document metadata (author, synopsis, etc.)
   - references
   - code objects (modules, classes, functions)
- Extensible with Python
